## Design Guidelines

### Color Scheme

- **Background**: Dark (black or dark gray) to match app theme
- **Text**: White with dark shadow/outline for readability
- **Accents**: Blue (#307DBE - app's primary color) or vibrant colors from palettes

### Typography

- **Primary Font**: SF Pro Display (iOS system font) or similar bold sans-serif
- **Title Size**: 72-90pt (for 6.7" display)
- **Subtitle Size**: 48-60pt
- **Weight**: Bold or Heavy for impact

### Layout

- **Text Placement**:
  - Top 10-15% or Bottom 10-15% of image
  - Center horizontally
  - Leave 5-10% margin on sides
- **Image Content**: Center 70-80% of screenshot
- **Safe Area**: Avoid text in corners (may be cropped)

### Text Overlay Best Practices

**Do**:
- ✅ Use short, punchy phrases (3-6 words)
- ✅ Focus on benefits, not features ("Transform Photos" not "Pixelation Algorithm")
- ✅ Maintain consistent style across all screenshots
- ✅ Use high contrast (white text on dark backgrounds)
- ✅ Add subtle shadow or outline for readability

**Don't**:
- ❌ Overcrowd with text
- ❌ Use more than 2 lines of text per screenshot
- ❌ Cover important UI elements
- ❌ Use hard-to-read fancy fonts
- ❌ Make claims you can't support

## Creating Split-Screen Comparisons

### Method 1: Figma/Canva

```
1. Create artboard 1290 x 2796px
2. Draw vertical line at center (645px)
3. Place "Before" image on left, crop to line
4. Place "After" image on right, crop to line
5. Add "Before" and "After" labels at top
6. Add main heading and subtitle
```

### Method 2: Photoshop

```
1. New document: 1290 x 2796px
2. Place both images as layers
3. Add layer mask to top image
4. Use gradient or hard line at center
5. Add text overlays
6. Export as PNG
```

### Method 3: iOS Shortcuts (Quick)

```
1. Use Shortcuts app
2. "Combine Images" shortcut
3. Select side-by-side layout
4. Add text with "Add Text to Image" action
```

## Testing and Optimization

### A/B Testing Strategy

Test two sets of screenshots:
- **Set A**: Feature-focused (show UI and options)
- **Set B**: Results-focused (show beautiful pixel art)

Metrics to track:
- Impression to product page views
- Product page views to downloads
- Conversion rate

Create localized versions for major markets:
- **English** (US, UK, Australia)
- **Korean** (South Korea)
- **Japanese** (Japan)
- **Spanish** (Spain, Latin America)
- **German** (Germany)

Translate text overlays and captions, keep screenshots the same.
