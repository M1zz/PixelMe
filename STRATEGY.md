# PixelMe 수익화 전략

## 목표

| 단계 | 기간 | 월 매출 |
|------|------|---------|
| 1단계 | 1~2개월 | ₩10만 |
| 2단계 | 3~4개월 | ₩30만 |
| 3단계 | 6개월+ | ₩50만 |
| 4단계 | 1년+ | ₩100만 (포트폴리오 합산 포함) |

---

## 1단계: 기반 (₩10만)

비용 ₩0. 전환율과 자연 유입만으로 도달.

### 가격 구조 정리
- 무료: 변환 3회/일, 워터마크 포함, 팔레트 1개
- Pro: 월 ₩4,900 / 연 ₩29,900
- 평생: ₩99,000 (또는 제거)

### 워터마크 바이럴 루프
- 무료 결과물에 작고 예쁜 "PixelMe" 워터마크
- Pro는 워터마크 없음 → 제거 욕구 = 전환 동기

### ASO 기본 최적화
- 키워드: 픽셀아트, 레트로 사진, 도트 변환, pixel art, retro filter
- 스크린샷: 비포→애프터 강조
- 부제: "사진을 픽셀 아트로 — 레트로 감성 필터"

---

## 2단계: 유입 (₩30만)

앱이 스스로 퍼지는 구조 만들기.

### SNS 원탭 공유
- 변환 완료 → 인스타/틱톡 공유 버튼
- 사이즈 프리셋 자동 적용 (스토리, 릴스, 피드)
- 비포/애프터 합성 이미지 자동 생성

### 프로필 픽셀화 모드
- 얼굴 감지 → 자동 크롭 → 정사각 출력
- "내 프사를 픽셀로" — 가장 공유되기 쉬운 콘텐츠

### 콘텐츠 시딩 (₩0)
- 유튜브 "개발자리"에서 인디 개발 스토리
- 인스타/틱톡에 결과물 주 2~3개
- 레딧 r/pixelart, 트위터 #pixelart 커뮤니티

---

## 3단계: 반복 수익 (₩50만)

구독자 누적 + 추가 수익원.

### 시즌 필터 팩
- 분기마다 1개 (₩1,900~3,900)
- "90년대 아케이드", "Y2K 감성", "지브리 풍" 등
- Pro 구독자는 전체 포함 → 구독 유지 동기

### 비포/애프터 영상 자동 생성
- 사진→픽셀 변환 과정을 2~3초 영상으로
- 틱톡/릴스 바로 업로드 포맷

### 구독 유지
- 월 1회 "이달의 무료 팔레트" 푸시
- 해지 시 "3일 더 무료" 리텐션 오퍼

---

## 4단계: 확장 (₩100만)

- 앱 포트폴리오 전략: PixelMe + Rereminder + LeaveWise + ClipKeyboard
- 또는 PixelMe 단독 (SNS 바이럴 시)
